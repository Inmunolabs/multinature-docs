# Minutas de Reunión

Este directorio contiene registros de reuniones operativas.
No es documentación de producto ni reglas de negocio es registro estructurado de reuniones operativas (ventas, producto, soporte, dirección).

**Propósito de las minutas:**

- Dejar trazabilidad de decisiones
- Registrar acciones claras con responsables
- Facilitar seguimiento y rendición de cuentas
- Servir como insumo para automatización futura (tickets, backlog)

👉 **Las minutas son eventos, no documentación estable.**

## Reglas

- Las decisiones se migran a su sección correspondiente
- Las acciones generan tareas o tickets
- Una minuta siempre vive en una subcarpeta temática
- El nombre siempre empieza por fecha
- Nunca se mezclan con planes, roadmaps o tareas

---

## 🧩 Template Oficial

Todas las minutas **deben** usar el siguiente template:

📄 [`./_TEMPLATE-minuta.md`](./_TEMPLATE-minuta.md)

No respetar el formato = la minuta pierde valor operativo.
